Ogród Zen CSS

Magia projektowania w CSS

Pokaz tego, co można osiągnąć dzięki projektowaniu w CSS. Wybierz dowolny arkusz stylów z listy, aby załadować go na tę stronę.

Pobierz przykładowy plik HTML i plik CSS

Droga do oświecenia

Na ciemnej i ponurej drodze walają się szczątki dawnych tagów specyficznych dla przeglądarki, niekompatybilnych DOM-ów, uszkodzonych CSS-ów i zarzuconych przeglądarek.

Lepiej zapomnijmy o tym, co było. Oświecenie w internecie nadeszło dzięki niestrudzonym wysiłkom postaci takich jak W3C, WaSP, czy twócy najważniejszych przeglądarek.

Ogród Zen CSS to miejsce relaksu i medytacji nad ważnymi naukami mistrzów. Przejrzyj na oczy. Zacznij korzystać z uświęconych tradycją technik w nowy i ożywczy sposób. Zjednocz się z siecią.

O co nam chodzi?

Ciągle warto pokazywać siłę CSS. Ogród Zen ma zachwycać, inspirować i zachęcać do uczestnictwa. Na początek oberzyj projekty z listy. Kliknięcie dowolnego z nich spowoduje załadowanie arkusza stylów na tę stronę. Plik HTML pozostaje ten sam, zmienia się jedynie zewnętrzny plik CSS. Niewiarygodne, prawda?

CSS daje pełną i całkowitą kontrolę nad stylem dokumentu hipertekstowego. Można to zobrazować w naprawdę zachęcający sposób, pokazując, co będzie, gdy powierzymy wodze tym, którzy potrafią nadać stukturze piękno. Do piękna sieci przyczynili się i projektanci, i programiści przyczynili się do piękna sieci - Ty też możesz wnieść swój wkład.

Udział

Naszym celem zawsze był atrakcyjny wygląd strony. Będziesz zmieniać tę stronę, więc znajomość CSS będzie oczywiście potrzebna, ale przykładowe pliki mają na tyle wyczerpujący opis, że mogą być punktem wyjścia nawet dla początkujących użytkowników CSS. Bardziej zaawansowaną wiedzę i wskazówki dotyczące pracy z programem CSS znajdziesz w przewodniku po zasobach CSS.

Możesz dowolnie modyfikować arkusz stylów, ale nie plik HTML. Jeśli nigdy wcześniej nie pracowałeś w ten sposób, możesz odczuć zniechęcenie, ale skorzystaj z podanych łączy, aby dowiedzieć się więcej, i użyj przykładowych plików jako wskazówki.

Pobierz przykładowy plik HTML i CSS i pracuj na kopii lokalnej. Po ukończeniu arcydzieła (proszę nie przesyłać niedokończonych prac) prześlij swój plik CSS na serwer sieciowy pod Twoją kontrolą. Wyślij nam link do archiwum tego pliku i wszystkich powiązanych zasobów, a jeśli postanowimy go wykorzystać, pobierzemy go i umieścimy na naszym serwerze.

Korzyści

Dlaczego warto wziąć udział? Aby uzyskać uznanie, znaleźć inspirację i stworzyć zasoby, dzięki którym można pokazać innym, jak niesamowity jest CSS. Ta witryna jest jednocześnie inspiracją dla osób już pracujących w sieci, narzędziem do nauki dla tych, którzy dopiero będą tworzyć strony, oraz galerią przyszłych technik, na które wszyscy czekamy.

Wymagania

W miarę możliwości korzystaj przede wszystkim z CSS 1 i 2. Stosowanie CSS 3 i 4 należy ograniczyć tylko do powszechnie obsługiwanych elementów, albo należy zadbać o silne zabezpieczenie na wypadek niepowodzenia. Ogród Zen CSS to funkcjonalność i praktyczność CSS a nie najnowsze i najnowocześniejsze sztuczki, które będzie mogło zobaczyć tylko jakieś 2% internautów. A tak naprawdę, jedyny nasz wymóg to poprawny arkusz CSS.

Na szczęście w tym trybie projektowania można zobaczyć, na ile skutecznie różne przeglądarki korzystają z CSS. Jeśli będziesz postępować zgodnie z wytycznymi, zobaczysz dość spójne wyniki w większości nowoczesnych przeglądarek. Ze względu na ogromną liczbę programów klienckich dostępnych obecnie w Internecie — zwłaszcza jeśli weźmie się pod uwagę urządzenia mobilne — nie na każdej platformie da się uzyskać układ idealny co do piksela. Tak już jest, ale przetestuj wszystkie dostępne wersje. Twój projekt powinien działać w co najmniej IE9+ i najnowszych wersjach rzeglądarek Chrome, Firefox, iOS i Android (a używa ich ponad 90% populacji).

Prosimy o przesyłanie oryginalnych prac graficznych. Prosimy o przestrzeganie praw autorskich. Ogranicz do minimum materiały budzące zastrzeżenia i spróbuj włączyć do swojej pracy unikalne i interesujące motywy graficzne. Raczej nie potrzebujemy już kolejnych projektów z wątkiem ogrodu.

To jest praktyka, a też pokaz umiejętności. Zachowujesz pełne prawa autorskie do swoich grafik (z ograniczonymi wyjątkami, patrz wskazówki dotyczące przesyłania), ale prosimy o udostępnienie Twojego pliku CSS w ramach licencji Creative Commons takiej samej, jak zamieszczona na tej stronie aby inni mogli uczyć się z Twojej pracy.
